
graphic
1. A symbol produced by a process such as handwriting, drawing, printing, engraving, etc. 2. An image in computer graphics format.

bullet
A large dot  used as a design element in text; often used to set off each item in a list. This kind of list is called a bulleted list.

Table
A presentation of information organized in rows and columns. 

Style:
A system for managing font faces and sizes within a document. Styles will have preset font attributes for different types of text including body text, headings of various levels, headers and footers, page numbers, and other types of text.

Locations of functions within the Word 2010 menus: 
bullet:
1. right click with the applications key, select the bullets split button
2. Select the bullets split button from the third group on the home ribbon

insert a picture:
insert ribbon, second group 

change margins:
page layout ribbon, second group
*As of this writing, a bug exists which prevents you from jumping directly to the second group with control right arrow. You will have to tab through the themes group to reach it 

change orientation of document:
page layout ribbon, second group
*As of this writing, a bug exists which prevents you from jumping directly to the second group with control right arrow. You will have to tab through the themes group to reach it 

print:
1. access the ribbon with alt, press left until you hear file tab, press down until you hear print tab, press right arrow
2. press control+P

center document vertically on page:
page layout ribbon, select page setup from second group, navigate to layout tab, tab to vertical alignment, select center

Keystrokes for Working in Tables
JAWS supports the same standard commands for working in tables on the internet, in Microsoft Word, and in most popular applications, allowing you access right away - without having to learn the  application specific commands. As you become more familiar with new applications, you may find it easier to use the commands specific to that
program, or it might be easier for you to continue using these commands.

Commands for Moving within Tables
These commands move focus to the indicated cell and speak the contents. Cell and row coordinates as well as column and/or row heading information may also
be provided.

Say Current Cell  ALT-CTRL-NUM PAD 5  
Cell to Right  ALT-CTRL-RIGHT ARROW  
Cell to Left  ALT-CTRL-LEFT ARROW  
Cell Below  ALT-CTRL-DOWN ARROW  
Cell Above  ALT-CTRL-UP ARROW  
First Cell  ALT-CTRL-HOME  
Last Cell  ALT-CTRL-END  
First Cell in Column  ALT-CTRL-SHIFT-UP ARROW  
Last Cell in Column  ALT-CTRL-SHIFT-DOWN ARROW  
First Cell in Row  ALT-CTRL-SHIFT-LEFT ARROW  
Last Cell in Row ALT-CTRL-SHIFT-RIGHT ARROW
